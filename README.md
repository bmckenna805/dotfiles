# [chezmoi](https://www.chezmoi.io/) Dotfiles Management


## Install

* [Install](https://www.chezmoi.io/docs/install/) the `chezmoi` binary
* `chezmoi init https://github.com/bmckenna805/dotfiles.git`
* `chezmoi apply`
